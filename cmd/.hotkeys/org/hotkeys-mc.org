|---------------+----------------------------------------------------------------------------|
| Hotkey        | Description                                                                |
|---------------+----------------------------------------------------------------------------|
| C-x i         | set other panel to information display mode                                |
| C-x q         | set other panel to quick view mode                                         |
| M-c           | cd command                                                                 |
| C-t           | tag file or directory                                                      |
| M-g           | select top file on the panel                                               |
| M-r           | select middle file on the panel                                            |
| M-j           | select bottom file on the panel                                            |
| +             | tag files by a pattern                                                     |
| \             | untag files by a pattern                                                   |
| M-i           | make directory of current panel the current directory of the another panel |
| M-s           | quick search in the current directory                                      |
| C-Space       | show directory sizes                                                       |
| C-\           | show directory hotlist                                                     |
| M-`           | show screens list                                                          |
|---------------+----------------------------------------------------------------------------|
| M-Shift-h     | show directory history                                                     |
| M-y           | move to previous directory in the directory history                        |
| M-u           | move to next directory in the directory history                            |
|---------------+----------------------------------------------------------------------------|
| M-Shift Enter | copy full path of file to command line                                     |
| M-Tab         | autocomplete filename, command, variable to command line                   |
|---------------+----------------------------------------------------------------------------|
